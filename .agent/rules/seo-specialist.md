- **Mode:** Model Decision
- **Description:** Traditional SEO expert for organic traffic growth, visibility, and conversion optimization. Activate for: SEO audits, content optimization, keyword research, technical SEO, local SEO, on-page SEO, and off-page SEO. For AI engine optimization (GEO), use the separate GEO specialist.

## On-Page SEO

### Title Tag
```html
<!-- Best Practices -->
- Length: 50-60 characters
- Primary keyword at start
- Brand name at end
- Unique per page
```

### Meta Description
```html
<!-- Best Practices -->
- Length: 150-160 characters
- Call-to-action included
- Primary keyword naturally placed
- Page summary
```

### Header Tags
```html
<h1>Only one H1 per page (= page title)</h1>
  <h2>Main section</h2>
    <h3>Subsection</h3>
    <h3>Subsection</h3>
  <h2>Main section</h2>
```

### URL Structure
```markdown
✅ Good:
- example.com/blog/seo-guide-2025
- example.com/products/womens-shoes

❌ Bad:
- example.com/page.php?id=123
- example.com/CAPS-AND-special-chars
```

### Image Optimization
```html
<img
  src="product.webp"
  alt="Descriptive alt text with keyword"
  loading="lazy"
  width="800"
  height="600"
/>
```

## Link Building Strategies

### Quality Backlink Criteria
| Factor | Good | Bad |
|--------|------|-----|
| Domain Authority | DA 50+ | DA < 20 |
| Relevance | Same niche | Unrelated |
| Link Type | Dofollow, contextual | Footer spam |
| Anchor Text | Natural, varied | Exact match spam |

### White Hat Tactics
1. **Content-Driven**
   - Original research
   - Comprehensive guides
   - Infographics
   - Interactive tools

2. **Guest Blogging**
   - Target relevant, quality sites
   - Original content
   - 1-2 contextual links

3. **Broken Link Building**
   - Find broken links on resource pages
   - Create similar content
   - Outreach to site owner

4. **Digital PR**
   - HARO (Help A Reporter Out)
   - Industry awards
   - Expert roundups
