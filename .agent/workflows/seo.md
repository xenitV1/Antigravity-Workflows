# SEO Workflow

## Description
Use this workflow to optimize content for traditional search engines like Google and Bing. It covers technical SEO audits, on-page optimization, keyword research, E-E-A-T content creation, and local SEO.

---

## Steps

1. **Analyze the SEO task** - Determine if the user needs an audit, content optimization, keyword research, local SEO, or technical improvements.

2. **Load SEO specialist rule** - Activate the @seo-specialist rule to apply SEO best practices and guidelines.

3. **Perform relevant checks based on task type**:
   - **For audits**: Check Core Web Vitals (LCP < 2.5s, INP < 200ms, CLS < 0.1), site structure, sitemap, robots.txt, canonical tags, and mobile-friendliness.
   - **For content optimization**: Review title tags (50-60 chars), meta descriptions (150-160 chars), heading hierarchy, URL structure, image alt text, and internal linking.
   - **For keyword research**: Identify seed keywords, use keyword tools, analyze competitors, and categorize by search intent (informational, commercial, transactional, local).
   - **For local SEO**: Verify Google Business Profile completeness, NAP consistency, local keywords, and citation sources.
   - **For new content**: Apply E-E-A-T principles (Experience, Expertise, Authoritativeness, Trustworthiness) and create topic clusters.

4. **Implement schema markup** - Add structured data: Article, Organization, Person/Author, LocalBusiness (for local SEO), FAQ, Breadcrumb, and Review schemas as appropriate.

5. **Review and validate** - Ensure all optimizations follow Google's guidelines, no keyword stuffing, no duplicate content, proper E-E-A-T implementation, and mobile-first design.

6. **Generate deliverable** - Provide audit report with prioritized recommendations, optimized content, keyword research report, or local SEO setup guide based on the task.

7. **Document metrics setup** - Recommend Google Search Console, Google Analytics 4, and keyword tracking tools for ongoing monitoring.
