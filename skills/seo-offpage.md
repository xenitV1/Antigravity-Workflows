# 1. Backlink Quality

## 1.1 Backlink Quality Criteria

| Criterion | Good Backlink | Bad Backlink |
|-----------|---------------|--------------|
| **Domain Authority** | DA 50+ | DA < 20 |
| **Relevance** | Same industry/niche | Unrelated site |
| **Traffic** | High organic traffic | Zero traffic |
| **Link Type** | Dofollow, contextual | Nofollow, footer/sidebar |
| **Anchor Text** | Natural, branded, keyword mix | Exact match spam |

## 1.2 Link Velocity

```markdown
Natural Link Growth:
- Monthly 5-10 quality backlinks (small sites)
- Monthly 20-50 backlinks (medium-scale)
- Avoid sudden spikes (spam signal)
- Consistent and steady growth
```

## 3.2 How to Disavow

```markdown

2. List spam links (disavow.txt)

4. Upload file

domain:another-spam.net

# Specific URLs

http://badsite.com/low-quality-link
```
