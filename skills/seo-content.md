# 1. Keyword Research & Strategy

## 1.1 Keyword Research Process

```markdown
1. Determine Seed Keywords
   - Main topics and categories
   - Product/service names
   - Industry terms

2. Use Keyword Tools
   - Google Keyword Planner
   - Ahrefs Keywords Explorer
   - SEMrush Keyword Magic Tool
   - Ubersuggest
   - AnswerThePublic

3. Competitor Analysis
   - Which keywords are competitors ranking for?
   - Where are content gaps?
   - Which keywords are winnable?

4. Search Intent Analysis
   - Informational (seeking information)
   - Navigational (seeking specific site)
   - Commercial (product research)
   - Transactional (purchase intent)

5. Keyword Filtering
   - Search volume vs. competition balance
   - Relevance
   - Keyword difficulty
   - CPC (commercial value indicator)
```

## 1.2 Keyword Types & Search Intent

| Keyword Type | Intent | Example | Strategy |
|--------------|--------|---------|----------|
| **Short-tail** | Navigational/Informational | "SEO" | High competition, brand awareness |
| **Mid-tail** | Commercial | "SEO tools" | Medium competition, consideration phase |
| **Long-tail** | Transactional | "best SEO tool 2025" | Low competition, high conversion |
| **Question** | Informational | "how to do SEO?" | Featured snippet opportunity |
| **Local** | Transactional | "Istanbul SEO expert" | Local pack, GBP optimization |

# 2. On-Page SEO

## 2.1 Title Tag Optimization

```html
<!-- ‚ùå Bad Examples -->
<title>Homepage</title>
<title>Welcome | ABC Ltd.</title>
<title>Cheap Shoes Sale Store Online Shopping Discount Campaign</title>

<!-- ‚úÖ Good Examples -->
<title>Women's Shoes Models 2025 | 50% Off | ABC Shoes</title>
<title>SEO Consulting Istanbul | Google Ranking | Mehmet SEO</title>

<!-- Best Practices -->
- Length: 50-60 characters (540px)
- Primary keyword at start
- Brand name at end
- Not clickbait, informative
- Unique title per page
```

## 2.2 Meta Description

```html
<!-- ‚ùå Bad Examples -->
<meta name="description" content="Welcome">
<meta name="description" content="shoes, sports shoes, boots, sandals, slippers, shoe models">

<!-- ‚úÖ Good Example -->
<meta name="description" content="2025 women's shoe models 50% off! Heels, sports, boots and sandals. Free shipping and easy returns. Discover now!">

<!-- Best Practices -->
- Length: 150-160 characters
- Includes call-to-action (CTA)
- Primary keyword naturally placed
- Summarizes the page
- Unique description per page
```

## 2.3 Header Tags (H1-H6)

```html
<!-- ‚úÖ Correct Hierarchy -->
<h1>How to Become an SEO Specialist? [2025 Guide]</h1>

  <h2>What is SEO?</h2>
    <h3>On-Page SEO</h3>
    <h3>Off-Page SEO</h3>
    <h3>Technical SEO</h3>

  <h2>Required Skills for SEO Specialist</h2>
    <h3>Technical Knowledge</h3>
    <h3>Analytical Competency</h3>

<!-- Best Practices -->
- Only 1 H1 per page
- H1 = Page title (same or very similar)
- Logical hierarchy (H2 > H3 > H4)
- Use keywords naturally
- Descriptive and informative
```

## 2.4 URL Structure

```markdown
‚ùå Bad URLs:
- example.com/page.php?id=123&cat=5
- example.com/a/b/c/d/e/f/very-long-url-structure
- example.com/CAPS-AND-special-chars

‚úÖ Good URLs:
- example.com/blog/seo-guide-2025
- example.com/products/womens-shoes
- example.com/istanbul/kadikoy-coffee-shop

Best Practices:
‚òê Use lowercase
‚òê English characters instead of special chars
‚òê Use hyphens between words (-)
‚òê Short and descriptive
‚òê Contains keyword
‚òê Remove stop words (and, or, with, for)
‚òê Use HTTPS
```

## 2.5 Image Optimization

```html
<!-- ‚úÖ Optimized Image -->
<img
  src="womens-shoes-2025.webp"
  srcset="womens-shoes-320.webp 320w,
          womens-shoes-640.webp 640w,
          womens-shoes-1024.webp 1024w"
  sizes="(max-width: 600px) 100vw, 50vw"
  alt="2025 women's shoe models - heels and sports"
  title="Women's Shoe Collection"
  loading="lazy"
  width="1024"
  height="768"
/>

<!-- Best Practices -->
‚òê Descriptive file name (womens-shoes.webp)
‚òê Always fill alt text
‚òê Use WebP or AVIF format
‚òê Responsive images (srcset)
‚òê Lazy loading active
‚òê Reduce file size with compression
‚òê Specify width & height (prevent CLS)
```

## 2.6 Internal Linking

```markdown
Strategies:

1. Hub & Spoke Model
   [Pillar Page: SEO Guide]
      ‚Üì ‚Üì ‚Üì
   [On-Page] [Off-Page] [Technical SEO]

2. Contextual Linking
   - Natural links within content
   - Descriptive anchor text
   - "Learn more about SEO tools" instead of "Click here"

3. Breadcrumb Navigation
   Home > Blog > SEO > On-Page SEO

4. Related Posts
   - Related articles widget
   - Recent posts

Best Practices:
‚òê Prevent orphan pages (pages with no internal links)
‚òê Anchor text variety
‚òê Deep linking (not just homepage)
‚òê Broken link check
‚òê Optimal link count (less than 2000)
```
